## Animation & Transitions

All transitions use:
- **Duration**: 0.3s
- **Easing**: ease-in-out
- **Properties**: transform, box-shadow, border-color

### Examples
- Hover lift: `transform: translateY(-4px)`
- Shadow on hover: `box-shadow: 0 12px 24px rgba(0,0,0,0.1)`
- Progress bar fill: Animated gradient

## Typography

- **Card Title**: Body 2, 500 weight, text-secondary
- **Main Value**: H4, bold, text-primary
- **Trend**: Body 2, 600 weight, success/error color
- **Progress Label**: Caption, text-secondary
- **Alert Title**: Subtitle, 600 weight
- **Alert Message**: Body 2, text-secondary

## Accessibility Features

1. **Tooltips**: Info icons with helpful explanations
2. **ARIA labels**: All interactive elements properly labeled
3. **Keyboard navigation**: Tab through all interactive elements
4. **Focus indicators**: Visible focus rings on keyboard navigation
5. **Color + Text**: Never rely on color alone (always include icons/text)
6. **Dismissible alerts**: Can be closed with keyboard (Escape key)
7. **Screen reader text**: Hidden text for context
