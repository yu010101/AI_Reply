## 🎯 テスト実行方法

```bash
# すべてのE2Eテストを実行
npm run test:e2e

# 特定のテストファイルを実行
npx playwright test e2e/auth.spec.ts

# UIモードで実行
npm run test:e2e:ui

# ヘッドモードで実行
npm run test:e2e:headed
```

## 📝 テスト実行前の準備

1. 開発サーバーが起動していることを確認（`npm run dev`）
2. 環境変数が設定されていることを確認（`.env.local`）
3. Supabaseの接続設定が正しいことを確認

## ⚠️ 注意事項

- テストはモックデータを使用しているため、実際のデータベースに影響しません
- 一部のテストは条件付きで実行されます（UI要素が存在する場合のみ）
- テストは並列実行されるため、テスト間でデータを共有しないように注意
